### 🔧 **Como Rodar o Projeto**
* Clone esse repositório.
* Crie um virtualenv com Python 3.
* Ative o virtualenv.
* Instale as dependências.
```
git clone https://github.com/flaviovl/ws-sigaa
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
